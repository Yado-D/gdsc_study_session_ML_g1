# Task 1 - File Descriptions

1st_Class.py — Builds a NumPy array from the numbers [11, 56, 80, 39, 56] and outputs basic statistics, including the average, the highest value, and the total of all elements.

numpy.ipynb — A Jupyter Notebook that walks through the same NumPy demonstration, broken into individual cells for importing the library, creating the array, running calculations, and showing the results interactively.

expense_tracking_ml.ipynb — A Notebook that showcases a simple machine-learning example for tracking expenses, covering steps such as preparing the dataset, training a basic model, and analyzing its performance.
